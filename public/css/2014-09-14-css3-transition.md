# css3 transition
> chapter 12

## write 
```css

transition <property> <duration> <timing-function> <delay>,
			<property> <duration> <timing-function> <delay>, 

```

### transition-property
> all or other property

### transition-duration
> duration 持续时间

### transition-timing-function
> timing-function

### transition-delay
> 默认为0
